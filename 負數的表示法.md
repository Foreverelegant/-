#負數的表示法
```
#1. 10010010
step1: [1]0010010最前面1是(負號) = -0010010  
    [1] 是負號  [0] 是正號
step2:010010  轉換  128 64 32 16 8 4 2 1  = 16+2 =18
step3:-18
```
```
#2.0010010
step1: [1]0010010  前面1(負號)  判斷正負 
    [1] 是負號  [0] 是正號
step2-1:計算大小
step2-2:先反向運算(inverse)
01101101 = 109
step3:完成 (正負號) + (大小)
-109
```
```
#3.0010010
step1:[1]0010010  前面1(負號) 判斷正負 
     [1] 是負號  [0] 是正號
step2-1:計算大小 10010001
step2-2:先反向運算(inverse)
 10010001  轉換   01101110 = 110
step3:-110
```
