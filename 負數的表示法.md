#負數的表示法
```
#1. 10010010
step1: 最前面1變(負號)= - 0010010  
step2:010010    128 64 32 16 8 4 2 1  = 16+2 =18
step3:-18
```
```
#2.
10010010
step1:前面1(負號) 判斷正負 
step2-1:計算大小
step2-2:先反向運算(inverse)
01101101 = 109
step3:完成 (正負號) + (大小)
-109
```
```
#3.
10010010
step1:前面1(負號) 判斷正負 
step2-1:計算大小 10010001
step2-2:先反向運算(inverse)
01101110 = 110
step3:-110
```
